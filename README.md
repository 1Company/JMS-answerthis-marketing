## Product Overview

**AnswerThis** is a real-time Q&A platform for professional seminars and live events.

**Key Features:**
- Silent, mobile-first question submission
- Moderator curation tools with keyboard shortcuts
- Speaker display with large, readable text
- No registration required for attendees

**Target Events:** Seminars, conferences, webinars, town halls, Q&A sessions

## Target Markets

- Corporate training departments
- Conference hosts
- Educational institutions

### Secondary

- Event management companies
- Speaker bureaus
